# بالعربية 

 الآن نحن نؤيد تعدد اللغات ! 

 تحميل حزمة اللغة التي تريدها ، إعادة تسمية " lan " و " main.exe " في نفس الدليل ، يمكنك استخدام اللغة المخصصة الخاصة بك ! 
 
 # 中文

 把你需要的语言包下载下来，重命名为“lan”，放到项目根目录即可使用！
